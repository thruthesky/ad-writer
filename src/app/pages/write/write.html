  • 글 작성 활용
    • 맨 밑에 링크 표시
      사이트 자체를 광고 할 때에는 별도를 글을 작성 할 필요는 없다. 예) 필고를 광고하면, 그냥 글 쓰고 맨 밑에 링크를 걸면 된다.
      URL 링크를 표시 할 때, 반드시 도메인을 텍스트에 포함해야 한다. 예) PhilGo www.philgo.com 이렇게 해야지만, 페이스북이나 기타 소셜에서는 HTML 태그를 모두 제거해야하는데, 이 때, A 태그에 포함된 URL 이 클릭 될 수 있다.
  • 네이버는 Blog API 로 태그를 지원하지 않는다. 공식문서에 "글 하단에 '태그: 필리핀,마닐라'" 와 같이 하고 있다고 한다. 따라서, 태그는 그냥 수동으로 하단에 입력을 한다.
  • 인터넷 속도가 느릴 때 가끔 에러가 난다.
  • 에러가 나는 경우, xxxxx-screenshot.png 파일을 확인하면, 에러가 난 화면을 볼 수 있다. 예를 들어, 티스토리에서는 브라우저 인증을 Hidden 모드에서 별도로 한번 해 주어야한다. 이러한 것은 screenshot.png 를 보면 금방 할 수 있다.
