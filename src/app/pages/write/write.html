  • 네이버는 Blog API 로 태그를 지원하지 않는다. 공식문서에 "글 하단에 '태그: 필리핀,마닐라'" 와 같이 하고 있다고 한다. 따라서, 태그는 그냥 수동으로 하단에 입력을 한다.
  • URL 링크를 표시 할 때, 반드시 도메인을 텍스트에 포함해야 한다. 예) PhilGo www.philgo.com 이렇게 해야지만, 페이스북이나 기타 소셜에서는 HTML 태그를 모두 제거해야하는데, 이 때, A 태그에 포함된 URL 이 클릭 될 수 있다.
